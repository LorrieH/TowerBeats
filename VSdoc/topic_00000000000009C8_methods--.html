Extension Methods
Name Description
DOBlendableColor(Color, Single) Tweens a Graphic's color to the given value, in a way that allows other DOBlendableColor tweens to work together on the same target, instead than fight each other as multiple DOColor would do. Also stores the Graphic as the tween's target so it can be used for filtered operations (Defined by ShortcutExtensions46).
DOColor(Color, Single) Tweens an Graphic's color to the given value. Also stores the image as the tween's target so it can be used for filtered operations (Defined by ShortcutExtensions46).
DOComplete(Boolean) Completes all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens completed (meaning the tweens that don't have infinite loops and were not already complete) (Defined by ShortcutExtensions).
DOFade(Single, Single) Tweens an Graphic's alpha color to the given value. Also stores the image as the tween's target so it can be used for filtered operations (Defined by ShortcutExtensions46).
DOFlip() Flips the direction (backwards if it was going forward or viceversa) of all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens flipped. (Defined by ShortcutExtensions).
DOGoto(Single, Boolean) Sends to the given position all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens involved. (Defined by ShortcutExtensions).
DOKill(Boolean) Kills all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens killed. (Defined by ShortcutExtensions).
DOPause() Pauses all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens paused. (Defined by ShortcutExtensions).
DOPlay() Plays all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens played. (Defined by ShortcutExtensions).
DOPlayBackwards() Plays backwards all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens played. (Defined by ShortcutExtensions).
DOPlayForward() Plays forward all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens played. (Defined by ShortcutExtensions).
DORestart(Boolean) Restarts all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens restarted. (Defined by ShortcutExtensions).
DORewind(Boolean) Rewinds all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens rewinded. (Defined by ShortcutExtensions).
DOSmoothRewind() Smoothly rewinds all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens rewinded. (Defined by ShortcutExtensions).
DOTogglePause() Toggles the paused state (plays if it was paused, pauses if it was playing) of all tweens that have this target as a reference (meaning tweens that were started from this target, or that had this target added as an Id) and returns the total number of tweens involved. (Defined by ShortcutExtensions).
